# Policy Module

Pure policy gates live here: source rights, attribution, retention, media-byte
storage, live-only restrictions, and future provider-specific constraints. The
module returns structured evaluations; it does not read files, call providers,
or write to targets.

Evaluates source rights, retention, attribution, media storage, live-only
constraints, and promotion eligibility. Policy decisions should be executable
and testable, not comments in adapters.
