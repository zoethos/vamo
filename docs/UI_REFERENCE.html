Design tokens — color & type

The locked palette. Lime is reserved for the single primary action on a screen (context-fab, Save expense, Settle up, Share Wrapped) — fill only, never lime text. Small accent text uses coral. Each plan-item type owns one accent (below).

Add to plan WAVE 2 type-first sheet · search & color cleanup

type-picker = type-tile ×6 (Visit/Train/Flight/Transfer/Lodging/Other)

Color rule applied: one accent per type (Visit = coral on selected tile, search icon, focus ring, suggestion highlight). Purple + teal borders removed; lime reserved for Save.

Plan view (itinerary) WAVE 2 the timeline Add-to-Plan feeds into

day-pills date selector (scrolls trip dates)
day-heading e.g. “Wed, May 14 — Ravello”
plan-timeline = time-rail + type-dot (spine) + plan-item-card
plan-item-card leading (poi-thumbnail OR type-icon-tile) + title + meta + rsvp-chips
poi-thumbnail place photo + corner type-badge (Visit/Lodging only)
distance-pill “400 m · 6 min walk” between two located stops
rsvp-chips + going-avatars Going / Maybe inline
add-to-plan-cta lime “Add to plan” → opens the type-picker sheet

Trip Map (journey replay) WAVE 3 a section inside a trip — see navigation map

member-trail per-member colored route, individually toggleable
trail-toggle chip per member (show/hide their trail)
map-pin avatar pin at a stop
moment-pin photo/expense pinned to the route
day-scrubber + replay-button drive the replay over trip days
Fed by Plan POIs + expense places + Memories → powers Trip Wrapped.

Trip Wrapped (recap story) WAVE 3 share-the-story payoff at trip close

wrapped-story tap-to-advance story format
story-progress-segments top progress bars (lime)
wrapped-slide cover / stat / distance / superlatives
share-wrapped-cta lime “Share your Wrapped” (carries watermark)
