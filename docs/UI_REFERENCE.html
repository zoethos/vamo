Vamo — UI Element Reference

Annotated wireframes of each screen with a canonical name for every element. Use these names (e.g. “hero-badge”, “participant-strip”, “total-spent-card”) to describe changes precisely. Mockups are schematic, not pixel-accurate — they map names to elements, not exact styling.

How to use: each screen shows numbered callouts ① ② ③ … on its elements; the legend beside it gives the canonical-name for each number. The Glossary at the bottom maps every name to what it is and the code file it lives in.

Updated 2026-06-09: unified "+" actions — every view uses a single top-right app-bar "+"; the bottom-nav center "+" and all section FABs were removed, bottom nav is now 4 items. Memories is a quick-action tile (grid is scrollable). The capture-carousel Background option now works (hero-background save fixed). Members' duplicate invite buttons removed.
